# 项目说明文档

## 项目目标
搭建一个"乡村建言平台"，分为“客户端”、“管理员端”以及“后端”

## 项目结构
- 参考sug中的代码，其中将admin开头的2个html文件，在rural_public_platform文件夹另外新建一个admin_frontend,
- 用户端在frontend的基础上改造，后端在backend的基础上改造
## 项目功能
- 支持分页查找
- 使用路由实现页面跳转
- 支持页面部分文本的编辑和收藏
- 后端的数据存储使用mongodb
- 必须用户注册/登录后，才能从实现页面跳转（JWT,全局前置守卫，权限控制、拦截器）
- 组件通讯，使用pinia记录user的信息，实现各组件的同步响应
- 后端使用中间件，解析用户凭证
- 统一使用用户凭证，通过axios在请求头添加
